# Contacto Page Override

Last updated: 2026-03-30

## Chosen Direction

`Split Editorial` with a small amount of `Premium Inquiry` restraint.

## Goals

- Make `Contacto` feel like a direct continuation of the `Home`
- Improve conversion clarity without making the page look template-like
- Use the split composition from the provided reference without copying it literally

## Composition

- Desktop opening block uses a dark / light split layout
- Left panel carries narrative, contact routes, and confidence cues
- Right panel carries the form on an off-white surface with navy accents
- FAQ continues below as a dark editorial section

## Visual Translation Rules

- Preserve the strong dark / light separation from the reference
- Translate display treatment into the site voice: `Anton` first, `Barlow Condensed` only as support
- Keep the form cleaner and more premium than the current version
- Use SVG icons from the existing icon system, never emoji icons
- Avoid the reference's custom cursor entirely

## Responsive Rules

- `>= 1024px`: split layout, with slightly heavier visual weight on the left
- `< 1024px`: stack content before form
- `< 640px`: single-column form rows and simplified quick-contact layout

## Motion

- Entrance reveals should feel premium, not flashy
- Card and CTA hovers should stay subtle and stable
- FAQ accordion should keep the current smooth behavior with reduced-motion fallback
